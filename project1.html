PROJECT TITLE

Autonomous Vehicle Navigation System Using Deep Learning and Computer Vision

Description

Developed a vision-based autonomous vehicle system for detecting and classifying real-time traffic objects like cars, pedestrians, and traffic signs. Trained and compared object detection models (YOLOv4 and Faster R-CNN) using the BDD100K dataset to support real-time decision-making in autonomous navigation.

Technologies Used

Python, OpenCV, TensorFlow, YOLOv4, Faster R-CNN, Google Colab, BDD100K Dataset

Brief Note

  • Built a real-time traffic object detection system using YOLO and Faster R-CNN.
  • Trained models on BDD100K dataset to identify vehicles, pedestrians, traffic signs.
  • Tracked object motion and calculated vehicle speed using OpenCV.
  • Compared model performance: YOLO (212 FPS, 18.6 mAP), Faster R-CNN (41.8 mAP).
